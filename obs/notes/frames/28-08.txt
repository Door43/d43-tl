===== Lahat ay Posible [28-08] =====

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-28-08.jpg?nolink&}}

Tinignan ni Jesus ang mga apostol at sinabi, “Imposible ito sa mga tao pero sa Diyos posible ang lahat ng bagay.”

===== Mga Mahahalagang Salita: =====

  * **[[:tl:obe:kt:jesus|Jesus,Jesus Christ, Christ Jesus]]**
  * **[[:tl:obe:kt:disciple|alagad, disipulo]]**
  * **[[:tl:obe:kt:god|Diyos]]**

===== Talaan para sa Pagsasalin: =====

  * **imposible ito sa mga tao**  - Ibig sabihin, “hindi maaaring magawa ng mga tao” o kaya “ang tao ay walang kakayahan para iligtas ang sarili nila.”
  * **sa Diyos posible ang lahat ng bagay**- Maaaring isalin ng ganito, “Kayang gawin ng Diyos ang lahat ng bagay, kahit ang pagliligtas sa isang mayamang tao” o “Kayang gawin ng Diyos ang mga imposibleng bagay, kung kaya't kaya niya ring iligtas ang isang mayamang tao.”
