===== Sumalakay ang mga Midianita [16-08] =====

{{https://api.unfoldingword.org/obs/jpg/1/en/360px/obs-en-16-08.jpg?nolink&}}

Sa kabilang dako naman bumalik ang mga Midianita na hindi mabilang sa dami para nakawan ang mga Israelita. Tinawag naman ni Gideon ang mga Israelita para labanan ang mga Midianita. Humingi ng dalawang palatandaan si Gideon sa Diyos para siguraduhin kung tutulungan nga siya ng Diyos para iligtas ang bayan ng Israel mula sa mga Midianita.

==== Talaan para sa Pagsasalin: ====

  * **Para nakawan ang mga Israelita **  – Maaaring isalin ng ganito, “ muling pumunta sa lupain ng mga Israelita para sila ay pagnakawan.”
  * **hindi mabilang**  - Maaaring isalin ng ganito, “ Napakarami ng mga Midianita para mabilang”
  * **dalawang palatandaan**  - Maaaring isalin ng ganito, “dalawang himala” o “ gawin ang dalawang imposibleng bagay.”
  * **tutulungan nga siya ng Diyos**  – Maaaring isalin ng ganito, “ Ibibigay ng Diyos ang kakayahang magawa ito” o kaya “tinatawag siya ng Diyos.”
  * **iligtas ang bayan ng Israel - **  Maaaring isalin ng ganito, “iligtas ang Israel sa mga Midianita.”
