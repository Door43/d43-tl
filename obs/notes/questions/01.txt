====== Kwento 1: Ang Kwento Tungkol sa Paglikha ======

===== Mga Tanong para sa Pag-unawa at Mga Sagot =====

  - Saan nanggaling ang lahat ng makikita sa buong kalawakan**?**
      * Nanggaling sa Diyos ang lahat// [01-01]//
  - Gaano katagal nagawa ng Diyos ang lahat**?**
      * Anim na araw// [01-01]//
  - Ano ang kadalasang tugon ng Diyos sa mga bagay na nagawa niya pagkatapos ng isang araw**?**
      * //[01-02] [01-04] [01-05] [01-06] [01-07] [01-08] //
  - Sa paanong paraan naiiba ang pagkakagawa ng Diyos sa tao kaysa sa mga hayop**?**
      * Ginawa ng Diyos na maging katulad niya ang tao //[01-04] //
  - Ano ang sinabi ng Diyos na magiging responsibilidad ng mga tao**?**
      * Pamumunuan ng mga tao ang mundo at at pangangalagaan nila ang mga hayop. //[01-05] //
  - Paano ginawa ng Diyos ang unang lalaki
      * Ginawa ng Diyos ang tao gamit ang alikabok//. [01-06] //
  - Paano nabuhay ang lalaki**?**
      * Hiningahan ng Diyos ng buhay ang lalaki. //[01-07] //
  - Ano ang pangalan ng lalaki**?**
      * Adam //[01-08] //
  - Saan inilagay ng Diyos si Adam**?**
      * Sa halamanan na itinanim ng Diyos.//[01-09] //
  - Sa anong natatanging puno hindi maaaring kumain si Adam**?**
      * Mula sa puno ng kaalaman ng mabuti at masama //[01-10] //
  - Ano ang mangyayari kapag kumain si Adam sa puno ng kaalaman ng mabuti at masama**?**
      * Mamamatay si Adam. //[01-11] //
  - Bakit “nag-iisa” si Adam gayong mayroon mga iba’t-ibang klase ng mga hayop**?**
      * Sagot: Hindi maaaring maging “katuwang” ni Adam ang mga hayop. //[01-12] //
  - Paano ginawa ng Diyos ang babae**?**
      * Ginawa ng Diyos ang babae mula sa tadyang ni Adam. //[01-13] //
  - Ano ang ibig sabihin ng “babae” para kay Adam**?**
      * Ang ibig sabihin nito ang ay galing ang “babae” sa “lalaki.” //[01-14] //
  - Ano ang kahalagahan ng pagkakaroon ng asawa ng lalaki
      * Sila ay magiging isa //[01-15] //
  - Ano ang naging pananaw Diyos ng sa lahat ng mga ginawa niya ng matapos na ang mga ito**?**
      * Nakita niyang napakaganda ng mga ito.// [01-16] //
  - Ano ang ginawa ng Diyos sa ikapitong araw**?**
      * Namahinga ang Diyos, pinagpala niya ang ikapitong araw at ginawa niya itong banal. //[01-17] //
