====== Kwento #4: Ang Pangako ng Diyos kay Abraham ======

===== Mga Tanong para sa Pag-unawa at Mga Sagot =====

  - Pagkatapos ng baha, napuno ba ng mga tao ang mundo gaya ng iniutos ng Diyos**?**
      * Hindi, nagtipon ang mga tao at nagtayo sila ng lungsod//. [04-01]//
  - Ilang uri ng wika ang mayroon sa mundo sa mga panahong iyon**?**
      * Iisa lang ang wikang ginagamit//. [04-01]//
  - Ano ang itinayong magkakasama ng mga tao sa halip na punuin nila ang mundo**?**
      * Nagsimula silang magtayo ng mataas na gusali para maabot ang kalangitan.// [04-02]//
  - Ano ang ginawa ng Diyos para magkahiwa-hiwalay ang mga tao sa buong mundo**?**
      * Pinalitan ng Diyos ang gamit nilang salita at ginawa niya itong marami at iba-iba.// [04-03]//
  - Ano ang pangalan ng lungsod na itinatayo nila**?**
      * Babel// [04-03]//
  - Ano ang ibig sabihin ng “Babel”**?**
      * Naguguluhan// [04-03]//
  - Ano ang ipinagawa ng Diyos kay Abram**?**
      * Sinabi ng Diyos kay Abram na iwan niya ang bansa niya at ang pamliya niya para pumunta sa ibang lupain.// [04-04]//
  - Ano ang ipinangakong gagawin ng Diyos para kay Abram**?**
      * Ipinangako niyang ibibigay niya kay Abraham ang mga lupaing nakikita nita, gagawin niya siyang dakila, gagawing niyang isang dakilang bansa ang mga kaapu-apuhan niya, at pagpapalain niya ang lahat ng mga pamilya sa mundo sa pamamagitan niya. //[04-04]//
  - Saang lupain pumunta si Abram**?**
      * Sa lupain ng Canaan.// [04-05]//
  - Anong pangako ang ginawa ng Diyos kay Abram tungkol sa lupain ng Canaan**?**
      * Ibibigay ito ng Diyos bilang pamana para kay Abram at sa mga kaapu-apuhan niya. //[04-06]//
  - Sino si Melchizedek**?**
      * Isa siyang lingkod na pari ng Diyos//. [04-07]//
  - Ano ang ginawa ni Melchizedek para kay Abram**?**
      * Pinagpala niya si Abram//. [04-07]//
  - Ano ang ibinigay ni Abram kay Melchizedek**?**
      * Nagbigay si Abraham ng ikapu ng lahat ng mayroon siya. // [04-07]//
  - Ano ang ipinangako ng Diyos kay Abram pagkatapos niyang tumira ng maraming taon sa Canaan**?**
      * Magkakaroon ng anak si Abram at magkakaapo siya ng kasingdami ng mga bituin sa kalangitan.// [04-08]//
  - Paano nasabi ng Diyos na matuwid si Abram**?**
      * Dahil naniniwala si Abram sa pangako ng Diyos.// [04-08]//
  - Ano ang “kasunduan”**?**
      * Ito ay pagsang-ayon ng dalawang panig sa isang bagay.// [04-09]//
