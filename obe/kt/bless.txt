====== pinagpala, pagpapala ======

===== Kahulugan: =====

Ang pagpapala ay ang mga mabuti at kapaki-pakinabang na bagay na mula sa Diyos. Ito ay maaaring pera, pagkain, kalusugan ng katawan at higit sa lahat ay buhay na walang hanggan.

  * Pinagpala ang tawag sa taong tumanggap ng mga pagpapala.
  * Ang kabaliktaran ng pagpapala ay “sumpa.”

===== Mungkahi sa Pagsasalin: =====

  * Ang salitang para "pagpalain" ay maaaring isalin ng ganito, "para pagkalooban ng marami" o "para itrato ng mabuti o pakitaan ng kabutihan"
  * Ang pangungusap na,"binigyan ng masaganang pagpapala ng Diyos" ay maaaring isalin ng ganito, "binigyan ng Diyos ng mga mabubuting bagay"
  * Ang "siya ay pinagpala" ay maaaring isalin ng ganito, "smakakaranas siya ng mga mabubuting bagay" o "gagawin siyang maunlad ng Diyos."
  * Ang "pinagpalaang taong" ay maaaring isalin ng ganito, "mabuti para sa isang tao ang"
